LJK Scanner Pro v7.5

Panduan Penggunaan Cerdas & Lengkap

Aplikasi LJK Scanner Pro adalah alat pemindai dan penganalisis Lembar Jawaban Komputer cerdas berbasis web. Aplikasi ini terbagi menjadi 3 menu utama di bagian bawah layar: Scan, Riwayat, dan Setelan.

Langkah 1: Persiapan Awal

Sebelum mulai memindai, pastikan Anda mengatur kelas dan kunci jawaban di tab Setelan.

1. Profil Kelas

  • Pilih/Buat Profil: Anda bisa menyimpan pengaturan spesifik untuk setiap kelas atau mata pelajaran menggunakan dropdown "Profil Kelas".
  • Simpan & Hapus: Klik ikon Centang Hijau untuk menyimpan profil, atau ikon Tempat Sampah Merah untuk menghapus profil yang sedang aktif.
  • Backup/Restore: Gunakan tombol Export Data (menyimpan semua data nilai ke file .json) dan Restore Data untuk mengembalikan/memindahkan data ke HP lain.

2. Kunci & Nilai (Sub-Menu)

  • Input Kunci (Maks 60 Soal): Masukkan deretan kunci jawaban tanpa spasi. Gunakan format khusus:
    X = Abaikan soal (soal dianulir).
    * = Bonus (semua jawaban dianggap benar).
    A/B = Opsi ganda (Contoh: A atau B dianggap benar).
  • Sistem Penilaian: Pilih metode perhitungan:
    - Jumlah Benar (Skor Mentah)
    - Persentase Standar (0 - 100)
    - Try Out UTBK (+4, -1, 0)
    - Bobot Soal (Bisa diatur bobot custom per nomor soal)
  • Batas KKM: Tentukan standar kelulusan kelas Anda (misal: 75). Ini akan menentukan warna Hijau/Merah pada hasil akhir.

3. Cetak LJK (Sub-Menu)

  • Kop Surat LJK: Anda bisa mengatur 4 Baris teks (Nama Ujian, Tahun Ajaran, Nama Sekolah, Alamat) dan mengunggah Logo Sekolah dari galeri HP.
  • Klik tombol Cetak LJK untuk menghasilkan file PDF LJK presisi yang siap diprint ke kertas A4.

4. Data Siswa (Sub-Menu)

  • Data Server (Supabase): Menarik nama siswa secara otomatis dari Cloud Database sekolah.
  • Upload Excel: Menggunakan file .xlsx untuk memasukkan daftar nama.
  • Ketik Manual: Masukkan data dengan format NIS=Nama Siswa per baris.

Langkah 2: Proses Pemindaian

Buka tab Scan untuk mulai mengoreksi kertas LJK.

1. Pengaturan Kamera

  • Auto-Scan: Jika aktif, kamera akan otomatis menjepret saat 4 kotak hitam di kertas LJK berhasil difokuskan. Jika gagal 5x, Auto-Scan akan berhenti otomatis.
  • Suara AI (TTS): Jika aktif, HP Anda akan membacakan Nama dan Nilai siswa secara otomatis setelah selesai memindai.

2. Proses Pemindaian & Rincian Hasil

  • Posisikan LJK: Pastikan 4 Kotak Hitam di sudut kertas masuk ke dalam layar dengan pencahayaan cukup.
  • Senter & Lensa: Gunakan ikon Senter untuk ruangan gelap. Jika kamera buram, gunakan ikon Tukar Lensa di sudut kanan atas untuk pindah ke lensa Ultra-Wide atau Macro.
  • Penyelamat Manual (Crop): Jika LJK terlipat, gelap, atau kamera gagal menemukan kotak hitam, sistem akan menawarkan Mode Manual. Anda cukup menggeser 4 titik merah mengambang ke tengah kotak hitam LJK secara manual!
  • Hasil Detik Itu Juga: Layar akan langsung menampilkan potongan scan, identitas siswa, Nilai Akhir, serta rincian Benar, Salah, Ganda, dan Kosong.
  • Rincian Tes: Di bawah nilai, terdapat tabel berisi rincian jawaban siswa (Tanda Centang Hijau, Silang Merah, atau Kosong). Jika siswa mendapat nilai 100, akan muncul efek ledakan Confetti! 🎉

Langkah 3: Riwayat & Analisis

Buka tab Riwayat yang memiliki 3 menu Sub-Tab (Sticky) di bagian atas.

1. Daftar Nilai

  • Menampilkan tabel ringkasan nilai seluruh kelas. Anda bisa mengurutkan data (Ranking) dengan mengeklik judul kolom (misal: klik kolom Skor).
  • Koreksi Manual: Jika ada coretan kotor yang salah terbaca, klik pada nama siswa di tabel untuk membuka popup Rincian Tes, lalu perbaiki centang jawabannya secara manual.
  • Aksi Individu: Klik ikon PDF (Merah) untuk mencetak Rapor Siswa, atau ikon WA (Hijau) untuk mengirim rincian nilai ke orang tua.
  • Aksi Massal: Di atas tabel, terdapat tombol Reset (Hapus Semua), Share WA Kelas, Print Rekap PDF, dan Download Excel.

2. Analisis

  • Dashboard Utama: Menampilkan Nilai Rata-rata, Tertinggi, Terendah, dan Grafik Donat yang memisahkan jumlah siswa Tuntas (Lulus) dan Remedial sesuai KKM.
  • Analisis Validitas Soal: Tabel pintar yang mendeteksi setiap butir soal. Menampilkan status Daya Pembeda (Baik/Cukup/Revisi) dan Tingkat Kesukaran (Sukar/Sedang/Mudah).

3. Nyontek (Radar Kecurangan)

  • Fitur investigasi canggih yang membandingkan jawaban antar siswa.
  • Sistem akan menyalakan alarm merah dan menampilkan nama pasangan siswa yang memiliki minimal 4 jawaban SALAH yang polanya identik persis. Semakin banyak salah yang sama, posisinya semakin di atas!

Fitur Ekstra & Pengaturan

  • Mode Gelap (Dark Mode): Klik ikon 🌙 di sudut kanan atas *header* untuk mengubah tema warna menjadi hitam elegan yang ramah di mata.
  • Install App (Bebas Kuota): LJK Scanner Pro adalah aplikasi PWA. Klik "Tambahkan ke Layar Utama" pada browser HP Anda agar aplikasi ini bisa diakses kapan saja meski tanpa jaringan internet!
