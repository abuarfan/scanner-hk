LJK Scanner Pro v6.7

Panduan Penggunaan Cerdas & Lengkap

Aplikasi LJK Scanner Pro adalah alat pemindai dan penganalisis Lembar Jawaban Komputer cerdas berbasis web. Aplikasi ini terbagi menjadi 3 menu utama di bagian bawah layar: Scan, Riwayat, dan Setelan.

Langkah 1: Persiapan Awal

Sebelum mulai memindai, pastikan Anda mengatur kelas dan kunci jawaban di tab Setelan.

1. Profil Kelas

  • Pilih/Buat Profil: Anda bisa menyimpan pengaturan spesifik untuk setiap kelas atau mata pelajaran menggunakan dropdown "Profil Kelas".
  • Simpan & Hapus: Klik ikon Centang untuk menyimpan profil baru, atau ikon Tempat Sampah untuk menghapus profil yang sedang aktif.
  • Backup/Restore: Gunakan tombol Export Data (menyimpan data ke file .json) dan Restore Data (memulihkan data dari file) untuk mengamankan data Anda.

2. Kunci & Nilai (Sub-Menu)

  • Input Kunci: Masukkan deretan kunci jawaban. Gunakan format khusus:
    X = Abaikan soal (soal dianulir).
    * = Bonus (semua jawaban dianggap benar).
    A/B = Opsi ganda (Contoh: A atau B dianggap benar).
  • Sistem Penilaian: Pilih metode perhitungan:
    - Jumlah Benar (Skor Mentah)
    - Persentase Standar (0 - 100)
    - Try Out UTBK (+4, -1, 0)
    - Bobot Soal (Bisa diatur bobot custom per nomor soal)
  • Batas KKM: Masukkan angka standar kelulusan kelas Anda (misal: 75).

3. Data Siswa (Sub-Menu)

  • Data Server (Supabase): Menarik data otomatis dari server utama jika tersedia.
  • Upload Excel: Menggunakan file .xlsx jika server sedang offline.
  • Ketik Manual: Masukkan data langsung dengan format NIS=Nama Siswa per baris.

4. Cetak LJK (Sub-Menu)

  • Anda bisa mengatur 4 Baris Kop Surat (Nama Ujian, Tahun Ajaran, Nama Sekolah, Alamat) dan mengunggah Logo Sekolah.
  • Klik tombol Cetak LJK untuk mencetak lembar kosong yang siap dibagikan ke siswa.

Langkah 2: Proses Pemindaian

Buka tab Scan untuk mulai mengoreksi LJK.

1. Pengaturan Fitur Scan

  • Auto-Scan: Jika aktif, kamera akan otomatis menjepret saat LJK berhasil terbaca oleh mesin.
  • Suara (TTS): Jika aktif, AI akan membacakan nama dan nilai siswa secara otomatis setelah berhasil di-scan.

2. Kontrol Kamera & Upload

  • Nyalakan Kamera: Tekan tombol utama untuk membuka aliran video dari kamera belakang.
  • Posisikan LJK: Pastikan 4 Kotak Hitam di sudut kertas LJK masuk ke dalam area bingkai putus-putus biru di layar.
  • Senter & Lensa: Gunakan ikon Senter untuk menyalakan lampu kilat di ruangan gelap. Jika HP Anda memiliki banyak kamera, gunakan ikon Tukar Lensa yang melayang di layar untuk mencari lensa yang paling fokus.
  • Upload Foto: Mengoreksi LJK dari foto yang sudah ada di galeri perangkat Anda.

Langkah 3: Melihat Hasil & Analisis

Setelah selesai memindai, buka tab Riwayat untuk melihat rekapitulasi kelas.

1. Daftar Nilai (Sub-Menu)

  • Menampilkan tabel ringkasan nilai yang bisa diurutkan (Sort) dengan mengeklik judul kolomnya.
  • Koreksi Manual: Jika ada kesalahan baca karena coretan kotor, klik pada baris nama siswa di tabel untuk mengubah jawabannya secara manual.
  • Aksi Per Siswa: Klik ikon PDF (Merah) untuk mencetak Rapor Individu, atau ikon WhatsApp (Hijau) untuk mengirim rincian nilai langsung ke HP siswa/orang tua.
  • Aksi Massal: Di bagian atas tabel, terdapat tombol untuk Reset data kelas, Share WA (Rekap Nilai Peringkat), Print (Rekap Tabel PDF), dan Download Excel.

2. Analisis (Sub-Menu)

  • Menampilkan dashboard visual berupa Nilai Rata-rata, Tertinggi, Terendah, dan Grafik Donat ketuntasan siswa.
  • Terdapat analisis butir soal mendalam seperti Tingkat Kesukaran (Sukar/Mudah) dan Daya Pembeda untuk mengevaluasi kualitas soal Anda.

3. Nyontek (Sub-Menu)

  • Fitur investigasi cerdas yang mendeteksi indikasi kecurangan dengan mencari persentase kesamaan jawaban salah di antara siswa.

Fitur Ekstra (Sangat Penting!)

  • Mode Gelap (Dark Mode): Klik ikon Bulan/Matahari di sudut kanan atas header untuk mengubah tema warna aplikasi agar nyaman di mata.
  • Install App (Offline Mode): Klik tautan "Install App" pada banner biru di bagian atas (jika muncul) agar aplikasi ini masuk ke Home Screen HP Anda dan bisa dijalankan super cepat tanpa kuota internet layaknya aplikasi asli!
